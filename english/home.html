Product Documentation

Icam Post

This documentation covers Icam Post. It describes the use of QUEST to develop post-processors as well as the post-processor itself, named GENER. It includes instructions on how to run GENER, detail on the post-processor vocabulary and detail about the extensive macro language available for advanced post development.

Virtual Machine

This documentation covers the ICAM Virtual Machine (VM) module. It explains the command Interface used with GENER and CERUN. It also describes how to use the views to manipulate the scene, how to create and load tooling, fixture, part and stock. A developer section for QUEST provide detail about creating the machine kinematic, importing and creating components, setting up the collision groups and using other features is explained.

CAM Interfaces

This documentation provide guidelines and instructions on how to set-up the interface between Icam Post GENER and Control Emulator CERUN with the supported CAM systems. Information about the CAM extractors used to transfer tooling, part, stock and CL file into ICAM modules is also provided.

User Interface

This documentation is a collection of specific information covering all ICAM module user interfaces. It includes help on dialog, view, menu, shortcut and so on. This section is mostly accessed via the context help "F1" key when using ICAM software user interfaces.
