Progress Window
---------------

GENER can be executed running with full interface, minimal interface or,
in a background mode. The Progress Window represent GENER minimal
interface. The window shows base execution statistics like number of
errors, current machining time, etc...

|image1|

The “\ **More info**\ ” button provides the access to the full GENER
interface. This interface can be accessed at any stage of the process.
Once the post processing is completed the “\ **View Listing**\ ” button
provides the access to the listing file.

The execution process can be terminated at any time by pressing the
“\ **Exit**\ ” button.

The minimal GENER interface is used when the “Fast” button on the “Play”
toolbar is selected or when the minimal interface is selected from the
GENER Launch panel “Show” selection.
