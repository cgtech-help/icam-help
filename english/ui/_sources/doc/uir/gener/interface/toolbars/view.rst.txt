View
----

|image1|

This tool bar contains buttons for different view options. They are
equivalent to the corresponding commands in the View menu. Pressing any
button will open or close the window. These buttons are:

-  Input window
-  Output window
-  Console window
-  Macro window
-  Diagnostic window

The two rightmost buttons toggle visibility of Virtual Machine windows:

-  Simulation window
-  Controller window
