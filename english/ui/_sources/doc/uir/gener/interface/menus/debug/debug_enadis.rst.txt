DebugÂ»Enable/Disable Breakpoint (Ctrl+F9)
-----------------------------------------

Enable/disable the breakpoint on the current line where the text cursor
is.

Used during debugging to enable or disable a breakpoint on the Listing
window line that currently has been selected. A disabled breakpoint
appears as a hollow red circle. GENER does not pause processing when it
encounters disabled breakpoints.
