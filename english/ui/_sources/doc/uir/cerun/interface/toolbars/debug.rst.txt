Debug
-----

|image1|

This tool bar contains buttons for different debug options. They are
equivalent to the corresponding commands in the Debug menu. These
buttons are:

-  Source Window
-  Variables Window
-  Stack Window
-  Breakpoint Manager
-  Macro Manager
-  Remove All Breakpoints
