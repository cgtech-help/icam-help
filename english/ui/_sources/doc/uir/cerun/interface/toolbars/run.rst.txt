Run
---

|image1|

This tool bar contains buttons for different operation modes. They are
equivalent to the commands in the Run menu. These buttons are:

-  Restart
-  Play
-  Pause
-  Step
-  Fast

The three rightmost buttons are Find options from the Tools menu:

-  Find...
-  Find Previous
-  Find Next
