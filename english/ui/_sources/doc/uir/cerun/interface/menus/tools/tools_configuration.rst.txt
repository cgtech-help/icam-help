Use this menu selection to activate the ICAM Configuration (CONFIG)
utility. Configuration changes made after processing has begun will only
take effect once the current program is rewound to the start or when
CERUN is next started. The CONFIG utility can also be run as a
standalone application, from the Windows Start menu or from the command
shell.
