-------------------------------

Add or remove a breakpoint on the current line where the text cursor is.

Used during debugging to set or clear a breakpoint on the Listing window
line that currently has been selected. A solid red circle appears in the
margin of the Listing window next to any MCD block or macro line that
has a breakpoint set. CERUN pauses processing when it encounters a
breakpoint.
