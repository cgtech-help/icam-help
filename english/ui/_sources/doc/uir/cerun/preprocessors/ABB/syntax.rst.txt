ABB Robot Pre-Processor Syntax
==============================

This document list the specific ABB Robot syntax supported by the
pre-processor.

--------------

**Feature Support:**

+-----------+---------------------------------------------------------+
| Feature   | Example                                                 |
+===========+=========================================================+
| TFRAME    | [[200, 0, 30], [1, 0, 0 ,0]]                            |
+-----------+---------------------------------------------------------+
| LOAD DATA | [0.8,[62, 0, 17], [1, 0, 0, 0], 0, 0, 0]                |
+-----------+---------------------------------------------------------+
| TOOL DATA | [ TRUE, [[200, 0, 30], [1, 0, 0 ,0]], [0.8,[62, 0, 17], |
|           | [1, 0, 0, 0], 0, 0, 0]]                                 |
+-----------+---------------------------------------------------------+
| ROBTARGET | [ [600, -100, 800], [0.707170, 0, 0.707170,0], [0, 0,   |
|           | 0, 0], [ 9E9, 9E9, 9E9, 9E9, 9E9, 9E9] ];               |
+-----------+---------------------------------------------------------+
