Open
----

|image1|

This dialog appears after selecting File \| Open Post from the menu.

**Post Name:** Name of the post processor to read from the database and
load into memory where it can be modified or reviewed.

**All posts in the selected database:** This window lists the names of
all posts in the selected database for reference. You can select a name
from this list, which will then change the *Post Name* entry.

**Select database:** This drop-down lists all databases that are open.
Use the *...* button to browse for the database if it does not appear in
the drop-down list.
