Spindle Table
-------------

|image1|

This dialog appears when creating or modifying a single spinlde
code/speed value pair in the spindle table.

**Code**: Enter an integer code that defines one of the unique spindle
speed values recognized by the control.

**RPM:** Enter the corresponding spindle speed value.

**OK:** Accepts this change to the spindle table.

**Cancel:** Ignores this changes to the spindle table.
