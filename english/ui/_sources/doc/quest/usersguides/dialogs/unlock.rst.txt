Unlock database
---------------

|image1|

Unlock a previously locked database.

**Password:** Type the password previously used to lock this database.

**Remove Password Protection:** If checked, then the password protection
will be remove for this database.

**OK:** Try to unlock the database.

**Cancel:** Cancel database unlock.
