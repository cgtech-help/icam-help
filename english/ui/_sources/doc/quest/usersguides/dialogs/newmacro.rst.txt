New Macro
---------

|image1|

This dialog appears when the "New" button is selected in the User
Defined Syntax Macros panel.

**Macro Name:** Enter the command name for the macro. The name can be
selected from the list of all known command names. The command name can
also be entered in **class:subclass** format. For example, **2000:\***
specifies a macro that matches any post processor command.
