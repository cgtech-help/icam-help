Spindle Table
-------------

|image1|

This dialog appears when adding or modifying spindle table information.
A spindle table defines the correlation between an integer code and the
actual spindle speed.

**New button:** Insert a new code/speed value into the list.

**Open button:** Edit a code/speed pair from the list.

**Delete button:** Delete a code/speed pair from the list.

**OK:** Saves changes to the spindle table.

**Cancel:** Ignores any changes made to the spindle table since this
dialog appeared.
