Copy
----

|image1|

This dialog appears when performing a Copy operation on a post in the
database.

**Copy post to:** Enter the name to use for the new copy of the post
processor. In this example, the post processor named *PP_DEMO,1* will be
duplicated and saved using the name *PP_DEMO_BACKUP*.
