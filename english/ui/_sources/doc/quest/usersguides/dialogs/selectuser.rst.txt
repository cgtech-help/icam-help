Select User
-----------

|image1|

Normally export files are only accessible by you. This dialog allows you
to create an export file that can be accessed by a selected user.

The list of users represents all customers you are licensed to exchange
or export post processors with. Contact ICAM if you would like the list
to be modified.

**Select User Name:** Select the user the export file is intended for.

**OK:** Accept selection,

**Cancel:** Cancel user selection and entire export action.
