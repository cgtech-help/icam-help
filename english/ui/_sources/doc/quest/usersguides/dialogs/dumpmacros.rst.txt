Dump Macros
-----------

|image1|

This dialog appears after selecting the Dump button in the User Defined
Syntax Macros section. It is used to copy (or dump) all user defined
macros to a file in ASCII (readable) format.

**File:** Specify the file to contain the macros being dumped. Select
*...* to browse for the file.

**All Macros:** Select this checkbox to output all user defined syntax
macros to the specified file. Clear this checkbox to restrict the macros
being output to just those that are selected in the scrolled list.
