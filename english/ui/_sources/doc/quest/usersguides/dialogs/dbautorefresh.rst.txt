Auto Refresh
------------

|image1|

Refreshes your database list at a given time interval.

You can also refresh your database list at any time by pressing F5.

**Refresh time interval:** Number of minutes after which an automatic
refresh should occur. Specify 0 for no automatic refresh.

**OK:** Accept new interval value.

**Cancel:** Cancel any changes made to the interval value.
