Comment View
------------

|image1|

The comment view is used to write notes about the post processor.

**Reset :** Used to revert back all the change to what it was when you
entered the section

**Cancel :** Exit the editor without saving the changes made to the
comments

**OK :** Exit the editor and save the changes made to the comments
