View
----

**Database**\ *(Ctrl+D)*\ **:** View all or the selected database in the
`Database View <../maininterface/workspaces/databaseview.html>`__.

**Details:** Show details in the database view.

**By Type...:** Specify criteria to limit the list of object in the
database view. (See `By Type <../dialogs/byposttype.html>`__ dialog)

**Graphical Questions**\ *(Ctrl+F8)*\ **:** Toggle between classical
`Questionnaire view <../maininterface/workspaces/questview.html>`__ and
the new `Graphical Questionnaire
view <../maininterface/workspaces/gqview.html>`__ in a question section.

**Increase X Margin**\ *(Ctrl+Right)*\ **:** Increase left/right spacing
between controls in graphical view.

**Decrease X Margin**\ *(Ctrl+Left)*\ **:** Decrease left/right spacing
between controls in graphical view.

**Increase Y Margin**\ *(Ctrl+Down)*\ **:** Increase top/bottom spacing
between controls in graphical view.

**Decrease Y Margin**\ *(Ctrl+Up)*\ **:** Decrease top/bottom spacing
between controls in graphical view.

**Previous axes page**\ *(Ctrl+Page-up)*\ **:** Display the previous
axes page in the controller axes page when more then one page is
required to display all axes.

**Next axes page**\ *(Ctrl+Page-down)*\ **:** Display the next axes page
in the controller axes page when more then one page is required to
display all axes.

**Status Bar:** Show or hide the status bar.
