VM Controller
~~~~~~~~~~~~~

Enables the display of the Virtual Machine Controller window if hidden
or minimized. Hides the Controller window if already active. This window
can also be toggled from the View toolbar.

The Controller window has tabs arrayed along the top to support the
following functions:

-  `Jogging model axes <vmcontroller_axes.html>`__
-  `Viewing and setting fixture
   compensation <vmcontroller_fixtcomp.html>`__
-  `Viewing and setting tool compensation <vmcontroller_toolcomp.html>`__
-  `Reviewing the simulation <vmcontroller_timeline.html>`__
