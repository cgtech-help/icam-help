~~~~~~~~~~~~~~~~~~~~~

Enables the display of the Virtual Machine Controller window if hidden
or minimized. Hides the Controller window if already active. This window
can also be toggled from the View toolbar.

The Controller window has tabs arrayed along the top to support the
following functions:

-  `Jogging model axes <../../windows/vmcontroller_axes.html>`__
-  `Setting fixture
   compensation <../../windows/vmcontroller_fixtcomp.html>`__
-  `Setting tool
   compensation <../../windows/vmcontroller_toolcomp.html>`__
-  `Replaying the
   simulation <../../windows/vmcontroller_timeline.html>`__
