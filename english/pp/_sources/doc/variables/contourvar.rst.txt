Contouring Variables
====================

.. _var_assist:
.. index:: Variables; $ASSIST

+------------------+-------------------------------------------------+
| **$ASSIST**      | ASSIST status. (OFF,AIR,OXYGEN,ARGON or NITRGN) |
|                  |                                                 |
|                  | Type: Keyword, Read/Write                       |
+------------------+-------------------------------------------------+

   This variable contains the current ASSIST status. It has a value of
   OFF, AIR, OXYGEN, ARGON or NITRGN. Note that the value of this
   variable is a minor word, not a numeric minor code value.
