.. _macro_functions:

Macro Functions
***************

This chapter lists all macro functions available for use in GENER macros. It should be noted that
functions can be used to retrieve a value only. Functions cannot be used on the left-hand side of an
assignment statement.

The first section summarizes the various functions by category.
