Installation and Upgrade
************************

This chapter describes the Icam Post installation and upgrade procedures.
Be sure to read the :ref:`Database Upgrade Procedure<database_upgrade>`)
if you have post-processors from a previous version of
the software and plan on modifying the post-processors using the new release.
