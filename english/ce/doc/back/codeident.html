Code Identifiers

Code identifiers are listed alphabetically by identifier name. The Group column lists the group number that the code identifier belongs. Group numbers are used to track the active code of a related group of codes. Group number zero is reserved for non-modal codes. The Order column lists the default processing order of the code. Group and Order can be changed along with other properties of each code using the Code Table customization feature of QUEST (see “Code Customization”). The Used column is checked if Control Emulator has built-in processing for that code.
