Siemens NX ICAM TO CAM-POST INTERFACE DOCUMENTATION

Overview

The NX ICAM Interface Kit is designed to accommodate differences in syntax between NX CLSF (CLSF_STANDARD or CLSF_ADVANCED) format and ICAM's post processing languages. Unigraphics APT commands in a cutter location source (".cls") file are input to CAD-APT and CAM-POST. The syntax for these statements are found in SIEMENS's GPM manual. See CAM-POST chapter "Post processor Vocabulary" for more information on CAM-POST supported syntax.

Only commands beginning with certain Unigraphics major words are translated. Other post processor statements are passed through or thrown out. Not all translations shown in this document may be used with all ICAM posts. For example, a CYCLE statement may be selected in NX and correctly reformatted by the interface, but the post-processor may not support that CYCLE type; a post processing error would result.

Unless otherwise indicated, only the minor words listed will be converted. Other minor words may cause an error.
