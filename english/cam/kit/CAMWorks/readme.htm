CAMWorks TO CAM-POST INTERFACE DOCUMENTATION

Overview

The CAMWorks Interface Kit is designed to accommodate differences in syntax between Geometric Ltd. CAMWorks CAD/CAM system and ICAM's post processing languages. The CAMWorks Interface Kit supports “.apt” aptsource files output.

The CAMWorks Interface Kit includes an insert.apt file containing special commands and a kit.mac file containing macros that, as much as possible, convert CAMWorks post-processor commands to CAM-POST format. The kit.mac macros are by default matched after any user-defined macros defined in the post-processor, so their effects can be easily overridden.
