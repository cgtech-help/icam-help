Cadra

Cadra TO CAM-POST INTERFACE DOCUMENTATION

Overview

The CADRA Interface Kit is designed to accommodate the differences in syntax between CADRA and ICAM's post processing languages. It sets the input format to "CADRA", sets the input card length to 100 characters after omitting first 12, and converts special CYLINDER/{ON|OFF} sequence into CAM-POST 3000,5000 class record sequence.

The VECTOR command is converted into GOTO, and DISPLY command to PPRINT.

The CONTRL command in the insert.apt file is used to remove first 12 characters form each CADRA CL line except comment lines.

If required, any special words should be added to the words.dat file.
