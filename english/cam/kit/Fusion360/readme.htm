Fusion 360 TO CAM-POST INTERFACE DOCUMENTATION

Overview

The Fusion 360 Interface Kit is designed to accommodate differences in syntax between Fusion's and ICAM's post processing languages. Post processing in Fusion 360 will generate a cutter location (CL) file to then run inside ICAM's post processing software.

Fusion 360 Post Processor File

A Fusion 360 post processor is included to output standard APT syntax when generating the CL file. This file must be added to Fusion's standard post processor location in order to be used.

The Fusion post processor can be used to:

  • Output APT-CL file
  • Run GENER for post processing
  • Run ICAM Extractor for Simulation
