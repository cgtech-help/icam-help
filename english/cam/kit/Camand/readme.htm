Camand

SDRC/CAMAX Camand TO CAM-POST INTERFACE DOCUMENTATION

Overview

The SDRC/CAMAX Camand Interface Kit is designed to accommodate differences in syntax between SDRC's and ICAM's postprocessing languages. SDRC Camand APT commands in an ASCII CLDATA format file are input to CAD-APT and CAM-POST. The syntax for these statements are found in the SDRC Camand manual. See CAM-POST chapter "Post-processor Vocabulary" for more information on CAM-POST supported syntax.

Two forms of conversion take place: Camand APT file text substitutions, and CAM-POST macro processing conversions.

APT Text Substitutions

The CAMAND.INS file causes the following substitutions:

  1. DEEPCL and DEEPBK keywords are converted to DEEP and BRKCHP
  2. All PPFUN/CAMAX commands are commented (i.e. not processed)
  3. A continued comment warning message is disabled. These messages may appear if a PARTNO command is omitted from the program. They are of no importance.

CAM-POST Macro Processing

  1. A zero (or near zero) UNITS scale factor is ignored.
  2. TOOLNO commands are replaced by LOADTL commands.
  3. CLDIST and SET commands that control wire guide positioning for 5-axis Wire EDM's are converted to PIVOTZ commands required by CAM-POST,
  4. LOAD/WIRE and UNLOAD/WIRE are converted to LOADTL commands.
  5. SYNCTR commands are converted to HEAD commands.
