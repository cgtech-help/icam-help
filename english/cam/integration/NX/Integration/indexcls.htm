CAM-POST to NX Integration Documentation

Overview

Thanks for choosing CAM-POST, the NC post processing Product from ICAM Technologies Corporation.

CAM-POST is a post processor generator embedded within the NX architecture to generate NC Data output. It can accommodate a wide variety of machine types having multiple axis path control.

NX end-users can post process their CLS files through the same graphical user interface (GUI), allowing a short-to-null learning curve.

The NX_CLS Interface Kit is designed to interpret commands in the NX CLSF files (.cls files generated using CLSF_STANDARD or CLSF_ADVANCED templates) for the purpose of post-processing them through CAM-POST, ICAM's post-processing software.

ICAM to NX Integration Setup Utility

The integration setup utility can be found in the Tools section of the Portal


Choose NX system(s) to setup


Select CAM system by checking the check-box or double-mouse-click on the already checked CAM system. Check the latest CAM-POST revision if multiple instances exists.

NX Setup Page

Note: You may need admin priviledges.

Several integration options are available.

  • GENER UI & CERUN UI affect how the sofware is launched. 'Full' will launch the complete runtime interface.
  • Setup for current user. is used to setup the installation folder for this user only. (Recommended)
  • Setup for any user of this machine. is used to setup the installation folder for all users. Admin rights are required.
  • CfgDir & PostDir is used to bring ICAM custom files to the correct NX directories. It should automatically be recognized.
  • Set Extractor is used to install the extractor running environment in NX. Please close the CAM system before clicking on this.
  • Install Post is used to create a post processor in NX based on the checked ICAM post(s) below.
  • ICAMCLSF<ICAM> is used to determine the output syntax format for the CLS file. (Recommended)
  • Interface kit should be selected based on your CAM system, except if you have a custom kit. NX_CLS is recommended.
  • Add default database to the list will add the default database when launching ICAM within the CAM software. The default database depends on the answer for the setup. If unchecked, you can select your own database location.
Press OK to save the settings.
