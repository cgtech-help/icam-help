CAM-POST to 3DEXPERIENCE Integration

Overview

Thanks for choosing CAM-POST, the NC post processing Product from ICAM Technologies Corporation.

CAM-POST is a post processor generator embedded within the CATIA/3DEXPERIENCE architecture to generate NC Data output. It can accommodate a wide variety of machine types having multiple axis path control.

This integration document is addressed to CATIA/3DEXPERIENCE end-users to help organize post processor usage through CATIA/3DEXPERIENCE settings and PPTables; describes the steps to install, upgrade and configure CAM-POST to run in an integrated CATIA/3DEXPERIENCE environment.

ICAM to CATIA/3DEXPERIENCE Integration Setup Utility

The integration setup utility can be found in the Tools section of the Portal


Choose 3DEXPERIENCE system(s) to setup


Select CAM system by checking the check-box or double-mouse-click on the already checked CAM system. Check the latest CAM-POST revision if multiple instances exists.

CATIA Setup Page

Note: You may need admin priviledges.

Several integration options are available.

  • GENER UI & CERUN UI affect how the sofware is launched. 'Full' will launch the complete runtime interface.
  • Setup for current user. is used to setup the installation folder for this user only. (Recommended)
  • Setup for any user of this machine. is used to setup the installation folder for all users. Admin rights are required.
  • Update ICAM-PAR DLLs is used to update the ICAM-PAR DLL when integrating with CATIA/3DEXPERIENCE. Use the path above to specify the path to the CATIA/3DEXPERIENCE application.
  • Interface kit should be selected based on your CAM system, except if you have a custom kit. 3DEXPERIENCE is recommended.
  • Add default database to the list will add the default database when launching ICAM within the CAM software. The default database depends on the answer for the setup. If unchecked, you can select your own database location.
  • Include demonstration PP and CE delivered with CATIA/3DEXPERIENCE is a special box to remove or include the demonstration post. (Unchecked is recommended)
Press OK to save the settings.

ICAM PPTables for CATIA/3DEXPERIENCE

There are two ICAM PPTable files in the default CATIA/3DEXPERIENCE PPTables director:

  • ICAM_INCH.pptable(imperial units)
  • ICAM_MM.pptable(metric units)

The default installation directory for the CATIA/3DEXPERIENCE PPTables is typically:
C:\Program Files\Dassault Systemes\BXX\win_b64\startup\Manufacturing\PPTables

3DEXPERIENCE Part Operation Setup

Set manufacture preference output as ICAM



In the Resource Configuration View, edit the NC resource of the machine to select your custom post processor and corresponding ICAM post-processor words table. You can also immediately set the NC data type to ISO.

Select NC data format as Point (X,Y,Z) for 3 Axis output or Axis (X,Y,Z,I,J,K) for Multi-Axis output. Set all other options as shown in the window on the right. If you have NURBS licensed post-processor, check "3D Nurbs interpolation" option if you want to use NURBS output.


Post Processing Options and Information

The NC data output from 3DEXPERIENCE through the ICAM post-processor can be executed either in silent mode or in a graphical display mode.


CATIA NC Manufacturing Default Settings

CATIA option menu: Tools > Options > Machining > Output
Set the post processor option to ICAM


CATIA V5 Part Operation Setup

In the CATIA PPR Tree, double click the Part Operation. In the dialog-window, click the Machine icon.


In the Machine Editor window, select the machine type, your custom post processor and the corresponding ICAM post processor words table. You can also immediately set the NC data type to ISO.


Select NC data format as Point (X,Y,Z) for 3 Axis output or Axis (X,Y,Z,I,J,K) for Multi-Axis output. Set all other options as shown in the window on the right. If you have NURBS licensed post-processor, check "3D Nurbs interpolation" option if you want to use NURBS output.
