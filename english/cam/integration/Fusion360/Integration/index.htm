CAM-POST to PowerMill Integration

Overview

Thanks for choosing CAM-POST, the NC post processing Product from ICAM Technologies Corporation.

This integration document is addressed to Fusion 360 end-users by proposing a help on integrating their ICAM post-processor database Fusion 360.

The ICAM Integration setup is an utility software specially designed for integrating ICAM post-processors into Fusion 360. Fusion 360 end-users can post process through the same graphical user interface (GUI), allowing a short-to-null learning curve for end-users. The users will have a choice to run GENER or only post an APT output.

ICAM to Fusion 360 Integration Setup Utility

The integration setup utility can be found in the Tools section of the Portal


Choose FeatureCAM system(s) to setup


Select CAM system by checking the check-box or double-mouse-click on the already checked CAM system will redirect the focus to the corresponding CAM setup page.
Or, select the CAM system setup page for modifying.

Fusion 360 Setup Page

Note: You may need admin priviledges.

Several integration options are available.

  • 260 represents the current ICAM version to integrate with. If a different version is needed, use the drop down menu to select.
  • Setup for this user. is used to setup the installation folder for this user only. (Recommended)
  • Setup for any user of this machine. is used to setup the installation folder for all users.
  • GENER UI & CERUN UI affect how the sofware is launched. 'Full' will launch the complete runtime interface.
  • Interface kit should be selected based on your CAM system, except if you have a custom kit.
  • Use default database will be the database use when launching ICAM within the CAM software. The default database depends on the answer for the setup. If unchecked, you can select your own database location.
  • PostDir is used to point the location the default location of local post processors for Fusion 360 CAM.
  • CreatePost will create a .CPS file to the location specified previously for the selected post processor.
  • If Packaged as single .CPS is checked, then all the post processors will be merged into one .CPS file.
Press OK to save the settings.

Fusion 360 Post Processor Setup

When posting the NC program in Fusion 360, ensure to select the correct post from the list.


In the post properties, under the ICAM Settings, you can choose to output the CL file, APT Only, or to launch GENER. Use the drop down menu to select your prefered option.
