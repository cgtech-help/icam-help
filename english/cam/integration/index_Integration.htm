Integration Utility

CAM Integration Setup Utility is able to integrate CAM-POST directly with some CAM systems. The purpose of the CAM integration is to provide the NC programmer with the ability, from within the CAM system, to associate an ICAM post-processor to the manufacturing process and to have the CAM system then run CAM-POST when the post-processing. A second purpose of the CAM integration is to provide the means to 'extract' manufacturing data (such as tools, holders, parts and fixtures) from the CAM system to be used with Virtual Machine simulation. For more information about the setup utility and its functionality, please select a CAM system!

Setup for

Setup for the current user is used to set up the installation folder for this user only. (Recommended) Setup for any user of this machine is used to set up the installation folder for all users. Admin rights are required. If the setup is completed for both the Current user and all users, by default, the current user is prioritized. To use the values set for all users, select the 'Prioritizing all users' setting.

Vericut Optimizer

When having license of both Icam Post and either Vericut Optimizer (VO) or Vericut Force optimization module, it is now possible to automate executing feedrate optimization before post-processing (on the cutter location data rather than the G-code). The ICAM setup utility used to integrate with CAM systems also now provide controls to browse for the Vericut Optimizer or Vericut Force installation folder.
