Overview

Thanks for choosing CAM-POST, the NC post processing Product from ICAM Technologies Corporation.

This integration document is addressed to Mastercam end-users by proposing an help on integrating their ICAM post processor database through Mastercam.

Mastercam end-users can post process their NCI files (via .PST files) through the same Mastercam graphical user interface (GUI) allowing a short-to-null learning curve for end-users.

ICAM to Mastercam Integration Setup Utility

The integration setup utility can be found in the Tools section of the Portal


Choose Mastercam system(s) to setup:


Select CAM system by checking the check-box or double-mouse-click on the already checked CAM system will redirect the focus to the corresponding CAM setup page.
Or, select the CAM system setup page for modifying.

Mastercam Setup Page

Note: You may need admin priviledges for the following step.

Several integration options are available.

  • GENER UI & CERUN UI affect how the sofware is launched. 'Full' will launch the complete runtime interface.
  • Setup for current user. is used to setup the installation folder for this user only. (Recommended)
  • Setup for any user of this machine. is used to setup the installation folder for all users. Admin rights are required.
  • Copy Dedicated Mastercam Control files is used to copy, from the provided Mastercam Control definition and Machine definition, files to your local Mastercam directory. Use the directory path above to specify both the application location and shared directory.
  • Set Extractor is used install the necessary DLL to enable the Extractor, only if you have simulation from ICAM.
  • Install Post is used to install the necessary DLL for ICAM to launch inside of Mastercam.
    This MUST be clicked with or without a post processor selected.
  • Interface kit should be selected based on your CAM system, except if you have a custom kit. Mastercam is recommended
  • Add default database to the list will add the default database when launching ICAM within the CAM software. The default database depends on the answer for the setup. If unchecked, you can select your own database location.
Press OK to save the settings.
