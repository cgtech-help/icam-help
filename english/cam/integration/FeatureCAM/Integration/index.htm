Overview

Thanks for choosing CAM-POST, the NC post processing Product from ICAM Technologies Corporation.

This integration document is addressed to FeatureCAM end-users by proposing an help on integrating their ICAM post-processor database through FeatureCAM.

The FeatureCAM setup is an utility software specially designed to integrate ICAM post-processors in FeatureCAM. FeatureCAM end-users can post process their cut files through the same FeatureCAM graphical user interface (GUI) allowing a short-to-null learning curve for end-users.

ICAM to FeatureCAM Integration Setup Utility

The integration setup utility can be found in the Tools section of the Portal


Choose FeatureCAM system(s) to setup

Select CAM system by checking the check-box or double-mouse-click on the already checked CAM system will redirect the focus to the corresponding CAM setup page.
Or, select the CAM system setup page for modifying.
