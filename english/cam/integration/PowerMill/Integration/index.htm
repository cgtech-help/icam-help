CAM-POST to PowerMill Integration

Overview

Thanks for choosing CAM-POST, the NC post processing Product from ICAM Technologies Corporation.

This integration document is addressed to PowerMill end-users by proposing an help on integrating their ICAM post-processor database through PowerMill.

The PowerMill setup is an utility software specially designed to integrate ICAM post-processors in PowerMill 2015/2016/2017. PowerMill end-users can post process their cut files through the same PowerMill graphical user interface (GUI) allowing a short-to-null learning curve for end-users.

ICAM to PowerMill Integration Setup Utility

The integration setup utility can be found in the Tools section of the Portal


Choose PowerMill system(s) to setup


Select CAM system by checking the check-box or double-mouse-click on the already checked CAM system will redirect the focus to the corresponding CAM setup page.
Or, select the CAM system setup page for modifying.

PowerMill Setup Page

Note: You may need admin priviledges.

Several integration options are available.

  • 240 represents the current ICAM version to integrate with. If a different version is needed, use the drop down menu to select.
  • Setup for this user. is used to setup the installation folder for this user only. (Recommended)
  • Setup for any user of this machine. is used to setup the installation folder for all users.
  • GENER UI & CERUN UI affect how the sofware is launched. 'Full' will launch the complete runtime interface.
  • Interface kit should be selected based on your CAM system, except if you have a custom kit.
  • Use default database will be the database use when launching ICAM within the CAM software. The default database depends on the answer for the setup. If unchecked, you can select your own database location.
Press OK to save the settings.

PowerMill PlugIn

ICAM's integration contains a PowerMill plugin for an integrated post processing method.
To enable plugins inside of PowerMill, click View Tab > Winow panel > User Interface > Vertical Plugin Window or Horizontal Plugin Window.


All available NC programs are shown in the selection box. Double-clicking an item will launch GENER for post processing.
